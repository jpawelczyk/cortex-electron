# Design System

Visual language and component patterns for Cortex.

> **Status:** To be defined. This document will cover colors, typography, spacing, components, and interaction patterns.

## Principles

- **Futuristic but functional** — Visual appeal matters, but never form over function
- **Dark-first** — Optimized for dark mode, light mode supported
- **Keyboard-first** — Everything accessible via keyboard
- **Information-dense** — Respect the user's screen real estate
- **Consistent** — Same patterns everywhere

## Tech Stack

- **Tailwind CSS** — Utility-first styling
- **shadcn/ui** — Component primitives (Radix + Tailwind)
- **CSS Variables** — Theme tokens
